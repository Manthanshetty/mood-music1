PROBLEM 1 — YouTube is broken (most critical)
Remove every single line of YouTube code from the entire project. Search for youtube, youtu.be, iframe, ytdl, youtube-dl, yt-search in all files and delete all of it.
Replace with JioSaavn API — completely free, no API key, no login needed.
Search endpoint:
GET https://saavn.dev/api/search/songs?query={TERM}&limit=20
Mood → search term mapping (use exactly these):

Happy 😊 → "happy upbeat hindi punjabi songs"
Sad 😢 → "sad emotional hindi songs dil"
Chill 😌 → "chill lo-fi kannada hindi relaxing"
Energetic ⚡ → "energetic workout rap hindi songs"
Romantic 💕 → "romantic love bollywood songs"
Focus 🎯 → "focus instrumental study calm music"

How to extract and play audio:

From API response, each song has downloadUrl array with {quality, url} objects
Pick quality === "high" first, fallback to "medium" then "low"
Play using HTML5 <audio> element — full free playback, no region blocks, no time limits
Strip HTML tags like <em> from song names and artist names using .replace(/<[^>]+>/g, '')
Artist names: song.artists.primary.map(a => a.name).join(", ")
Duration: song.duration is in seconds — format as Math.floor(s/60) + ":" + String(s%60).padStart(2, "0")
Album art: song.image array — use highest resolution available
